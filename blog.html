What is DOM and what is the purpose of DOM?

DOM full meaning is "Document Object Model". It is a programming interface that allows us to create elements, remove elements and change elements from the document. We can also add events to make our webpage more dynamic.

How will you select HTML elements using DOM? Name the DOM methods.

There are five ways to select HTML elements using dom 1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

What is event bubble?

Event bubbling is a way of event propagation in the HTML DOM. The event bubble points an event to its target. When a user clicks a button, the even bubble fires an event targeting that button via event.target

What is a callback function and why will you use it?

Callback function is a function that passed into another function as an argument. And is called when the parent function has finished its execution. We use callback function because JavaScript actions are asynchronous. This allows for better control flow and flexibility in executing code.
